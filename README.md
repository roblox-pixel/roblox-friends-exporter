
# Roblox Friends Exporter

Outil Python permettant de récupérer et exporter la liste des amis publics d’un utilisateur Roblox via l’API officielle.

## Fonctionnalités
- Récupération des amis Roblox via API publique
- Gestion de la pagination automatique
- Récupération des noms et display names
- Export en fichier texte sur le bureau

## Installation

```bash
pip install -r requirements.txt

ROBLOX FRIENDS EXPORTER - INSTALLATION

========================================
            🪟 WINDOWS 
========================================

1. Télécharger Python :
https://www.python.org/downloads/

2. Lancer l’installation

IMPORTANT :
- Cocher "Add Python to PATH"
- Choisir "Install for me only"

3. Ouvrir CMD et vérifier :
python --version

ou :
py --version

4. Installer les dépendances :
pip install requests

Si ça ne marche pas :
py -m pip install requests

5. Lancer le script :
python main.py


========================================
🍎 MAC (macOS)
========================================

Méthode 1 (recommandée - Homebrew) :

1. Installer Homebrew (si pas installé) :
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2. Installer Python :
brew install python

Méthode 2 (sans Homebrew) :

1. Télécharger Python :
https://www.python.org/downloads/

2. Installer le fichier macOS normalement

3. Vérifier :
python3 --version

4. Installer dépendances :
pip install requests
ou
python3 -m pip install requests

5. Lancer le script :
python3 main.py
